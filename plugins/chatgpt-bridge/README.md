# ChatGPT Bridge plugin

This directory is the distributable Codex plugin.

The plugin opens ChatGPT in the Codex in-app browser, verifies that the user is signed in, coordinates single-image or batch-image generation, exports original page assets, and writes measured, review-ready manifests.

It does not use an OpenAI API key or collect credentials. Users sign in directly on `chatgpt.com` inside the in-app browser. See the repository [README](../../README.md) and [login guide](../../docs/LOGIN.md) for installation and privacy details.
