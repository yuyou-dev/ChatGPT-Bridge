# Contributor Instructions

- Keep the repository free of credentials, cookies, session tokens, temporary ChatGPT asset URLs, and user-specific absolute paths.
- Keep the plugin identifier `chatgpt-bridge`; use `ChatGPT Bridge` only as the display/project name.
- Treat the Codex in-app browser as the only supported browser surface.
- Never automate password, passkey, CAPTCHA, or two-factor entry. Pause for the user to complete those steps.
- Keep generated/exported/reviewed/approved states separate.
- Run `npm run check` before proposing a release.
