# ChatGPT Bridge site

Promotional website for the open-source
[ChatGPT Bridge](https://github.com/yuyou-dev/ChatGPT-Bridge) Codex plugin.

## Local development

Requires Node.js `>=22.13.0`.

```bash
npm install
npm run dev
npm run build
```

The site is built with Next.js-compatible Vinext and prepared for OpenAI Sites.
Production deployment is intentionally separate from preview/version creation.
